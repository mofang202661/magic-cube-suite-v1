# magic-cube-suite-v1
## 魔方套件-V1
魔方系列全套Windows桌面工具箱，纯本地运行独立EXE程序，正式稳定版V1.0

## 一、全套工具核心功能（共6款独立软件）
1. **文稿排版魔方**
批量文稿格式化、思源宋体统一排版、段落自动规整，适配小说书稿、网文批量排版需求。
2. **极速魔方**
本地线索批量采集、Excel文档高速处理、素材批量导出、图文素材自动整理工具。
3. **极速鉴卡主程序**
硬件显卡参数读取、算力检测、显卡型号批量识别、硬件信息导出存档工具。
4. **游戏魔方**
游戏画质参数优化、进程资源调控、帧率稳定优化、N卡显卡资源智能分配。
5. **矩阵魔方_视频消重V2.0**
短视频批量去重、画面裁切、字幕批量生成、多素材混剪预处理。
6. **系统优化魔方V3.0**
Windows系统轻量化优化、后台进程清理、磁盘资源调度优化。

## 二、硬件&系统适配
- 操作系统：Windows 10 / Windows 11 64位
- 推荐硬件：i5及以上CPU、16G/32G内存、独立N卡主机
