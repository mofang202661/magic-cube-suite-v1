# magic-cube-suite-v1
## 魔方套件-V1
魔方系列全套Windows桌面工具箱，包含文稿排版魔方、极速魔方、游戏魔方、矩阵视频消重魔方，正式版本V1.0

## 一、全套工具核心功能
1. 文稿排版魔方：批量文稿格式化、字体统一、段落自动规整、思源宋体批量排版，适配小说/书稿排版需求。
2. 极速魔方：本地线索采集、批量文件处理、文档极速导出、批量素材整理工具。
3. 游戏魔方：游戏画质优化、进程调控、帧率稳定、显卡资源分配优化。
4. 矩阵魔方_视频消重V2.0：短视频批量去重、画面裁切、字幕批量生成、素材混剪预处理。

## 二、硬件&系统适配
- 系统：Windows 10 / Windows 11 64位
- 推荐配置：i5及以上处理器、16G/32G大内存、N卡独显主机

## 三、版本说明（重要）
1. 本仓库仅提供演示程序，完整解锁功能需付费激活；
2. GitHub内EXE仅开放基础预览功能，无完整批量处理权限；
3. 全套完整版工具，可前往淘宝、拼多多、抖音店铺下单，一机一码硬件激活，永久使用。
